What is NEON-SOUL?

Every OpenClaw agent starts with a hand-written SOUL.md. You guess at its personality. You manually update it when things change. And every time you write "be helpful and honest," you know it means nothing — because the agent didn't earn those words.

NEON-SOUL changes that. Your agent's soul grows from its own experience.

The Old Way

You write a SOUL.md by hand. You guess what your agent should value. The personality is static — it never changes unless you change it. Your agent is whoever you told it to be.

The NEON-SOUL Way

Your agent interacts with users. Memory files accumulate. NEON-SOUL reads those interactions, finds patterns, and surfaces the values your agent actually demonstrates. When the same behavior shows up three times, it becomes a core belief. The soul writes itself.

Why CJK?

Those aren't decorative characters. CJK (Chinese-Japanese-Korean) characters are among the densest writing systems on earth — a single character like èª  (honesty/sincerity) encodes what takes an entire English sentence. Combined with emoji indicators and mathematical notation, one line like ğŸ’ èª : honesty > performance carries the same semantic weight as a full paragraph of prose.

This is how NEON-SOUL achieves 10x compression. Not by removing meaning — by using a writing system that packs more meaning per token.
