# NEON-SOUL Workflows

Operational processes for the NEON-SOUL project.

## Available Workflows

| Workflow | Purpose | Status |
|----------|---------|--------|
| [Documentation Update](documentation-update.md) | Systematic doc updates when architecture changes | Draft (N=0) |
| [Skill Publish](skill-publish.md) | Publish to npm and ClawHub registries | Active (N=3) |

## Workflow Lifecycle

```
Draft (N=0) → Pilot (N=1-2) → Active (N≥3) → Mature (N≥5)
```

- **Draft**: Initial creation, untested
- **Pilot**: Being tested, gathering observations
- **Active**: Proven useful, actively maintained
- **Mature**: Stable, rarely changes
